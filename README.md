Небольшой самописный framework для разработки сайтов. Имеет 
- адаптированные функции AJAX( jQuery),
- классы для работы с базами данных,
- удобные классы (деревья) для построения запросов в очередь,
- класс для работы с базами данных,
- адаптированные функции для работы с файлами и директориями,
- классы для создания контроллера  
- при помощи @media и css-grid выполняется адаптация сайта под различные разрешения устройства.
